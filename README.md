## About

TurboCamp 是向 BaseCamp 致敬的專案管理網站.

TurboCamp 主要功能如下：

- 新增屬於自己團隊的專屬專案
- 團隊內即時溝通
- 用行事曆管理當前專案的行程進度
- 將檔案上傳供團隊成員取用

## Technologies

Frontend: HTML / CSS（ SCSS 、 TailWind ） / JavaScript（ Stimulusjs ）
Backend: Ruby(3.1.1) / Ruby on Rails(6.1.5.1)
Database: PostgreSQL
Deployment: Heroku
Capistrano Version Control: Git / GitHub
第三方登入: Github、Google
檔案上傳: AWS S3
